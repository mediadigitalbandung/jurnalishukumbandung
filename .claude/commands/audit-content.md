# Audit-Content — Content Quality Audit Specialist

Deep audit kualitas konten: artikel hukum, duplikasi, non-legal, outdated, broken. Read-only.

## Input
$ARGUMENTS — scope: `quality`, `topical`, `duplicate`, `outdated`, `broken`

## Scope

- Kualitas editorial artikel
- Topical relevance (legal-only policy)
- Duplicate/near-duplicate content
- Outdated articles (facts may have changed)
- Broken media references (image 404)
- Article completeness (excerpt, image, tags, category)
- Auto-generated article quality

## Checklist

### CRITICAL (melanggar brand identity)
1. **Non-legal articles published** — artikel tech/auto/lifestyle yang published
2. **Duplicate articles** — judul atau konten sama > 90% similarity
3. **Broken featured image** — image 404 di artikel published
4. **No author assigned** — artikel tanpa authorId

### HIGH (kualitas editorial)
5. **Thin content** — < 300 kata
6. **No excerpt** — excerpt kosong atau default
7. **No tags** — artikel tanpa tags
8. **No category** — uncategorized
9. **Missing SEO metadata** — seoTitle/seoDescription kosong
10. **No verification label** — artikel tanpa UNVERIFIED/VERIFIED tag
11. **Auto-generated & unedited** — isAutoGenerated=true & updatedAt = createdAt

### MEDIUM (content health)
12. **Outdated articles** — > 6 bulan tanpa update, highly viewed
13. **Low-quality auto-generated** — published auto-article (should be draft)
14. **No source citations** — no `source` record linked
15. **Inline images broken** — content has `<img>` with 404 src
16. **HTML bloat** — inline styles, unnecessary tags

### LOW
17. **No internal links** — artikel tanpa link ke artikel lain
18. **Short title** — < 30 char (mungkin tidak descriptive)
19. **Long title** — > 100 char
20. **No alt text** di inline images

## Metodologi

```typescript
// 1. Non-legal detection
const NON_LEGAL_KEYWORDS = ["smartphone", "mobil", "motor", "iPhone", "gadget", "resep"];
const nonLegal = await prisma.article.findMany({
  where: {
    status: "PUBLISHED",
    OR: NON_LEGAL_KEYWORDS.map(k => ({ title: { contains: k, mode: "insensitive" } })),
  },
});

// 2. Duplicate detection
const byTitle = await prisma.$queryRaw`
  SELECT title, COUNT(*) FROM "Article"
  WHERE status = 'PUBLISHED'
  GROUP BY title HAVING COUNT(*) > 1
`;

// 3. Incomplete articles
const incomplete = await prisma.article.findMany({
  where: {
    status: "PUBLISHED",
    OR: [
      { excerpt: null },
      { excerpt: "" },
      { featuredImage: null },
      { categoryId: null },
      { tags: { none: {} } },
    ],
  },
});

// 4. Outdated
const sixMonthsAgo = new Date();
sixMonthsAgo.setMonth(sixMonthsAgo.getMonth() - 6);
const outdated = await prisma.article.findMany({
  where: {
    status: "PUBLISHED",
    publishedAt: { lt: sixMonthsAgo },
    viewCount: { gte: 1000 },
  },
});

// 5. Auto-generated unedited (published)
const autoPublished = await prisma.article.findMany({
  where: { isAutoGenerated: true, status: "PUBLISHED" },
});
```
